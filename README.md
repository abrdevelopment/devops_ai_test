# DevOps + IA – Pipeline de Diagnóstico Assistido

Este projeto demonstra como integrar IA de forma simples e confiável
em um pipeline DevOps usando GitHub Actions.

## Objetivo
- Executar deploy
- Coletar logs
- Executar análise assistida por IA
- Falhar o pipeline apenas após o diagnóstico

## Benefícios
- Diagnóstico aparece mesmo em falha
- Pipeline auditável
- Evolução natural para AIOps / ChatOps

## Conceito-chave
> A IA não decide o pipeline. Ela explica o erro.
